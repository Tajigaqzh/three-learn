# 中国史读书路径与诗词格律笔记

本文整理自一组关于中国史读书路径、经典阅读、典故积累和诗词格律的对话。它不是完整教材，而是一份便于继续阅读、写作和扩展的路线图。

## 阅读索引

这份笔记可以按三条线来读：

- **中国史阅读线**：三代与《诗经》、春秋战国、秦汉与东汉、前四史、魏晋南北朝、隋唐、五代十国、宋辽夏金、元、明、清、晚清近代、民国。
- **方法线**：从人物入手，继续追问集团、制度、资源和历史后果；从故事阅读转向问题阅读。
- **诗词附章**：诗体、平水韵、中华新韵、七绝七律、粘对、拗救、孤平、三平尾、宋词与词律，以及典故积累。

如果只想快速进入历史阅读，可以从第一节“三代与《诗经》入门”开始，按朝代顺读到民国；如果目标是写旧体诗词，可以直接跳到末尾“附章：诗词格律、词律与典故积累”。

## 一、三代与《诗经》入门

传统说“三代”，通常指夏、商、周。再往前的三皇五帝，半在神话、半在传说政治谱系里。通行本《史记》开篇是《五帝本纪》，不是司马迁原作的“三皇本纪”；常见的《三皇本纪》多出自唐代司马贞《补史记》。所以读这段觉得懵，是正常的：神话、传说、礼制想象和早期历史本来就混在一起。

读三代，不要一开始就只问“尧舜禹是否真实存在”。第一步可以先弄清楚：传统中国人心里的三代，是怎样一套贤王、贤臣、礼制和天命叙事。

### 三代贤者

先抓核心八人：

```text
尧、舜、禹、汤、文王、武王、周公、伊尹
```

再扩展：

```text
皋陶、傅说、太公望、召公、伯夷叔齐、比干、箕子、微子
```

这些人物在传统里各有象征：

- **尧**：禅让、任贤、忧民，是儒家理想君主的开端。
- **舜**：孝德、受禅、举贤，常与尧并称“尧舜”。
- **禹**：治水、勤劳、建立夏代，被称“大禹”。
- **汤**：商的开国君主，放桀救民，是“革命”叙事里的贤王。
- **文王**：周族兴起的德主，仁厚、能聚贤。
- **武王**：伐纣建周，传统上被视为顺天应人。
- **周公**：三代贤者核心人物之一，制礼作乐、辅成王、安定周初。
- **伊尹**：辅佐商汤灭夏，后来又有“放太甲”故事，是贤相典型。

贤臣和节义人物也很重要：

- **皋陶**：舜、禹时代贤臣，传统上与刑法、公正有关。
- **伯益**：助禹治水，也与早期部族和礼制传说有关。
- **傅说**：商王武丁贤相，出身版筑，被举用后辅政。
- **太公望 / 姜子牙**：辅佐文王、武王，周初开国谋臣。
- **召公**：周初重臣，与周公并称，辅佐成王、康王。
- **伯夷、叔齐**：不食周粟，饿死首阳山，是出处进退和守节典故。
- **箕子、比干、微子**：殷商遗贤和忠臣典型，常用于商末政治和亡国叙事。

读这些人物时，可以问三件事：

```text
这个人物在传统里代表什么德行？
这个故事服务于什么政治观念？
后世诗文如何用这个典故？
```

例如“尧舜”常代表理想政治，“禹”代表治水与勤王事，“伊尹”代表贤相辅君，“周公”代表礼制与辅政，“伯夷叔齐”代表守节与不仕。

### 从《诗经》《大学》进入三代

《诗经》和《大学》是进入三代观念的两扇门：

- 《诗经》保存了周人对祖先、祭祀、农事、战争、婚姻和政治秩序的记忆。
- 《大学》虽然成书与整理较晚，但它讲的“修身、齐家、治国、平天下”，是后世理解三代政治理想的一套框架。

《大学》的核心链条：

```text
格物、致知、诚意、正心
→ 修身
→ 齐家
→ 治国
→ 平天下
```

这不是三代史料本身，但它说明后世儒家如何理解“三代之治”：政治不是只靠法令和权术，而是从君子的德性、家国秩序、礼乐教化层层展开。

### 《诗经》里的周人记忆

《大雅》可以重点回看：

- 《文王》：看周人如何塑造文王之德。
- 《大明》：文王、武王、天命、伐商。
- 《绵》：周族先祖迁徙和兴起。
- 《皇矣》：周族受命、文王征伐。
- 《生民》：后稷神话，周人农业祖先记忆。
- 《公刘》：周族迁徙经营。
- 《既醉》《凫鹥》：宗庙祭祀和福禄观念。

《周颂》可以重点看：

- 《清庙》：祭文王，宗庙气象很重。
- 《维天之命》：天命与文王。
- 《我将》：祭祀文王。
- 《时迈》：武王巡守、告祭。
- 《武》：武王克商。
- 《臣工》《噫嘻》《丰年》《载芟》《良耜》：农事、祭祀、王政。

这些诗不是单纯抒情，它们保存的是周人的政治宗教世界。

### 国风的读法

《诗经》分“风、雅、颂”。郑、秦、卫等都属于国风。国风最值得读的地方，是它保留了各地不同的声音：地方风俗、人情声音和起兴方法。

读国风不要只按“美刺”经学框架读，也不要只当爱情诗读。最好同时看三层：

```text
字面情境：诗里发生了什么？
地方风俗：它呈现了什么社会生活？
后世解释：毛诗、朱熹、现代学者如何理解？
```

郑风气质明亮、流动、近人情。重点篇：

- 《将仲子》：女子拒绝情人翻墙而来，情感和礼法拉扯。
- 《叔于田》：写男子出猎，带有爱慕和观赏。
- 《子衿》：思念之诗。
- 《野有蔓草》：邂逅情诗，清新动人。
- 《溱洧》：三月上巳男女游春，风俗感很强。

秦风气质劲健、质朴、边地感强。重点篇：

- 《车邻》：车马威仪。
- 《驷驖》：田猎、车马、武力气象。
- 《小戎》：女子思征夫，车马兵甲写得很细。
- 《蒹葭》：意境迷离。
- 《无衣》：同仇敌忾，慷慨激昂。

卫风气质复杂、婉转，既有情爱也有讽世。重点篇：

- 《氓》：弃妇叙事诗，写婚恋、怨悔、人情变化。
- 《伯兮》：思妇念远征丈夫。
- 《硕人》：写庄姜美貌，铺陈华美。
- 《淇奥》：赞美君子。
- 《木瓜》：投赠酬答。

可以这样比较：

```text
郑风：男女游春、情感鲜活
秦风：车马兵甲、尚武劲健
卫风：婚恋世态、讽刺深婉
```

其他国风也值得按气质读：

- **周南、召南**：温厚、端正、礼俗感强。重点看《关雎》《葛覃》《卷耳》《桃夭》《芣苢》《鹊巢》《采蘩》《草虫》《行露》《摽有梅》。
- **邶风、鄘风**：哀怨、复杂、世态感重。重点看《柏舟》《绿衣》《燕燕》《击鼓》《谷风》《静女》《墙有茨》《君子偕老》《载驰》。
- **王风**：东周衰微，怀旧低沉。重点看《黍离》《君子于役》《采葛》《大车》。
- **魏风**：质直、讽刺、社会感强。重点看《伐檀》《硕鼠》《葛屦》《汾沮洳》。
- **唐风**：古朴、忧思、含蓄。重点看《蟋蟀》《山有枢》《扬之水》《绸缪》《鸨羽》。
- **陈风**：艳丽、迷离，有巫风感。重点看《宛丘》《东门之枌》《月出》《泽陂》。
- **齐风**：机警、讽刺、贵族社会感。重点看《鸡鸣》《还》《著》《东方未明》《南山》《甫田》。
- **曹风、豳风**：曹风有小国衰微和讽刺感；豳风保存农事岁时和古厚气息。重点看《候人》《蜉蝣》《下泉》《七月》《鸱鸮》《东山》。

如果按优先级读：

```text
第一层：《周南》《召南》《邶风》《卫风》《王风》《秦风》《豳风》
第二层：《郑风》《魏风》《唐风》《陈风》
第三层：《齐风》《曹风》《鄘风》《桧风》
```

如果按名篇路线：

```text
《关雎》《桃夭》《静女》《击鼓》《氓》《黍离》
《伐檀》《硕鼠》《蒹葭》《无衣》《月出》《七月》《东山》
```

### 三代阅读顺序

可以先这样走：

```text
《史记·五帝本纪》《夏本纪》《殷本纪》《周本纪》
→ 回看《诗经·大雅》《周颂》相关篇
→ 读《尚书》选篇
→ 再读《左传》里引用三代的段落
```

《尚书》可以先读这些：

- 《尧典》：尧舜政治理想。
- 《皋陶谟》：德、刑、政治伦理。
- 《禹贡》：九州、贡赋、地理秩序。
- 《汤誓》：商汤伐夏。
- 《盘庚》：商王迁都与政治动员。
- 《牧誓》：武王伐纣。
- 《金縢》：周公辅政与周初政治。
- 《召诰》《洛诰》：成周营建与周初制度。
- 《无逸》：周公论君主不可安逸。

读每篇时先问三个问题：

```text
它塑造了什么政治理想？
它讲了哪种君臣关系？
后世会怎样引用它？
```

这层通了，再进入甲骨文、金文、西周制度和现代夏商周史研究，才不会乱。

## 二、春秋战国阅读路径

春秋战国不适合一上来硬啃《史记》。它的材料分散在本纪、世家、列传、年表里，人物、国家和事件互相穿插，很容易读散。

如果此前主要通过《大秦帝国》三部曲、《东周列国志》、B 站历史视频和《史记》零散篇章了解这段历史，下一步要做的是：把影视和讲述里的故事，转成清楚的时间线、国家结构和制度变化。

### 总体抓法

春秋和战国要分开看。

春秋重点：

```text
礼崩乐坏
诸侯争霸
晋楚竞争
齐桓晋文
秦穆公、楚庄王
吴越争霸
贵族政治与宗法秩序松动
```

战国重点：

```text
三家分晋
田氏代齐
各国变法
合纵连横
兼并战争
秦国胜出
郡县制、军功爵、官僚国家兴起
```

一句话说：

```text
春秋看礼制崩坏下的诸侯秩序；
战国看兼并战争中官僚国家的形成。
```

### 入门通史

先读通史骨架，不要直接从《左传》《战国策》《史记》硬进。

- **顾颉刚、童书业《国史讲话：春秋》**：适合从电视剧、视频转到书本阅读，先建立春秋大势。
- **童书业《春秋史》**：春秋断代史经典，重点看诸侯争霸、礼制崩坏、晋楚竞争、华夷关系。
- **杨宽《战国史》**：战国入门和进阶都绕不开，重点看三家分晋、变法、合纵连横、兼并战争、秦统一。
- **吕思勉《先秦史》**：范围更宽，从上古到战国都有，适合看社会、制度、文化整体变化。

推荐先这样走：

```text
顾颉刚、童书业《国史讲话：春秋》
→ 童书业《春秋史》
→ 杨宽《战国史》
→ 吕思勉《先秦史》相关部分
```

### 春秋原典

春秋原典首选《左传》，但《左传》不建议从第一页硬读。

- **杨伯峻《春秋左传注》**：读《左传》的硬书，注释扎实，但很厚。适合按事件查读、精读。
- **高士奇《左传纪事本末》**：很适合作为入口。它把《左传》的编年材料改成事件专题，比直接读《左传》顺很多。

可以按这些事件和人物线读《左传》：

```text
郑庄公
齐桓公
晋文公
秦穆公
楚庄王
城濮之战
崤之战
弭兵之会
吴越争霸
```

读春秋时要特别注意：

- 盟会、会盟、朝聘背后的诸侯秩序。
- 卿大夫家族如何侵蚀国君权力。
- 晋国六卿、鲁国三桓这类贵族政治。
- “礼”不仅是礼貌，而是政治秩序和身份结构。

### 战国原典

战国材料可以先读《战国策》和《史记》重点篇，不要通读。

- **《战国策》**：看纵横家、游说、谋略、人性和政治语言。可用何建章《战国策注释》。
- **《史记》春秋战国相关篇章**：按国家和人物挑读。

《史记》可先读：

- 《秦本纪》
- 《六国年表序》
- 《赵世家》
- 《魏世家》
- 《韩世家》
- 《田敬仲完世家》
- 《商君列传》
- 《苏秦列传》
- 《张仪列传》
- 《廉颇蔺相如列传》
- 《白起王翦列传》
- 《孟尝君列传》
- 《平原君虞卿列传》
- 《魏公子列传》
- 《春申君列传》
- 《吕不韦列传》

读战国时要从人物故事继续往外追问：

```text
这个国家靠什么制度变强？
它的军事和财政基础是什么？
它为什么胜出或失败？
这个人物代表哪一种政治技术或时代变化？
```

例如读商鞅，不只看变法和车裂，还要看军功爵、县制、户籍、什伍、农战如何改变秦国。读苏秦、张仪，不只看口才，还要看合纵连横背后的国家利益和地缘格局。

### 专题深入

等通史和重点原典有了基础，再读专题研究。

- **杨宽《战国史料编年辑证》**：进阶工具书，把战国史料按年代整理，对解决《史记》《战国策》年代混乱很有用。
- **赵鼎新《东周战争与儒法国家的诞生》**：不是讲故事，而是解释春秋战国战争、国家形态、儒法传统如何塑造后世中国。

如果继续深入，还可以按问题线索扩展：

- 春秋贵族政治与宗法制度。
- 战国变法与官僚国家。
- 军功爵、郡县制、户籍和赋役。
- 纵横家与国际关系。
- 儒、墨、道、法、名、兵诸子思想。
- 秦国制度为何胜出。

### 思想书的衔接

春秋战国不只是政治史，也是诸子百家的时代。读历史时可以同步选读思想经典，但不要一口气全啃。

可以先按问题读：

- **孔子与礼乐秩序**：《论语》。
- **孟子与王道政治**：《孟子》。
- **荀子与礼法结合**：《荀子》选篇。
- **墨子与兼爱、尚贤、非攻**：《墨子》选篇。
- **老庄与对礼法秩序的反思**：《老子》《庄子》内篇。
- **法家与战国国家技术**：《商君书》《韩非子》选篇。
- **兵家与战争组织**：《孙子兵法》。

不要把诸子只当抽象哲学。它们都是在春秋战国政治、战争、社会秩序崩解和重建的背景下说话。

### 推荐阅读顺序

```text
顾颉刚、童书业《国史讲话：春秋》
→ 童书业《春秋史》
→ 杨宽《战国史》
→ 《左传纪事本末》
→ 《史记》春秋战国相关世家列传
→ 《战国策》
→ 杨伯峻《春秋左传注》按事件精读
→ 赵鼎新《东周战争与儒法国家的诞生》
→ 杨宽《战国史料编年辑证》等专题工具书
```

### 从影视和视频转入读书

《大秦帝国》和《东周列国志》适合作兴趣入口，但读书时要做校准：

- 电视剧为了戏剧冲突，会强化人物性格和台词。
- 历史视频能帮你建立事件线，但不能替代原典和研究书。
- 《史记》文学性很强，也有材料取舍和叙事结构，不宜只按故事读。

比较稳的方法是：

```text
先用通史建立大势
再用《左传纪事本末》《通鉴》式材料串事件
然后回到《左传》《战国策》《史记》重点篇
最后用专题研究解释制度和结构
```

春秋战国读进去以后，很多后世历史都会变清楚：秦汉制度、儒法关系、中央集权、士人政治、边疆战争、甚至后世王朝的兴亡逻辑，都能在这里看到源头。

## 三、秦汉与西汉阅读路径

秦汉史不要只按“刘邦打败项羽、汉武帝打匈奴”来读。更好的抓法，是把它看成从战国国家到统一帝国的形成过程。

可以先抓三条主线：

```text
秦如何把战国制度推成帝国
楚汉如何在秦制废墟上重组天下
汉如何在郡县、封国、儒法、财政、边疆之间完成长期稳定
```

如果此前主要通过课本、电视剧、《楚汉传奇》《大风歌》《刘邦传》《大汉帝陵》，以及详细历史视频了解秦汉，下一步要做的是：把熟悉的人物和事件，放进原典、制度和专题研究里重新读一遍。

### 已有基础的判断

如果已经熟悉以下线索，就不算秦汉零基础：

```text
楚汉争霸
→ 刘邦建汉
→ 吕后维持政权与外戚安排
→ 文帝从代王入继大统
→ 文景休养生息与功臣、诸侯王问题
→ 七国之乱
→ 武帝中央集权、开疆拓土
→ 对匈奴战争与西域经营
→ 财政危机和盐铁、均输平准等财政重组
→ 推恩令削藩
→ 巫蛊之祸与武帝晚年政治
→ 昭帝、霍光辅政
→ 宣帝中兴
→ 元成以来外戚、儒生、宦官与皇权问题
→ 平帝、王莽前夜
```

这已经是一条比较完整的西汉政治史框架。下一步不是再听更多故事，而是把视频叙事型知识升级为原典和问题驱动型知识。

可以概括为：

```text
叙事框架较完整
制度关键词已建立
原典阅读和专题研究还需要补
```

### 西汉主线怎么转成问题

把熟悉的故事转成问题：

```text
刘邦为什么能胜过项羽？
汉初为什么不能直接废封国、全行郡县？
吕后政治为什么能稳定，又为什么难以延续？
文帝为什么能从代王变成皇帝？
功臣集团如何制约皇权？
七国之乱为什么是郡国并行的转折点？
汉武帝为什么必须解决诸侯、匈奴、财政三件事？
长期战争为什么必然带来财政制度重组？
盐铁、均输平准是救急之策，还是国家能力扩张？
巫蛊之祸暴露了什么宫廷、继承和酷吏政治问题？
轮台诏是否意味着武帝政策转向？
霍光辅政为什么能稳定政局？
宣帝为什么能完成昭宣中兴？
西汉后期为什么外戚、儒生、灾异政治问题突出？
王莽为什么能以“周公再世”的形象获得合法性？
```

### 通史入门

- **吕思勉《秦汉史》**：断代史经典，前半按政治事件讲，后半按社会经济文化分门别类讲。适合建立秦汉整体框架，但文字略老派。
- **王子今《秦汉史：帝国的成立》**：更现代，结合传世文献和考古资料，适合从影视、课本转入正式阅读。
- **田余庆《秦汉魏晋南北朝简史》**：简明、严谨，适合先打底；以后再读田余庆论文集。

推荐顺序：

```text
王子今《秦汉史：帝国的成立》
→ 吕思勉《秦汉史》选读
→ 《史记》秦亡、楚汉、汉初人物线
→ 《汉书》本纪 + 食货志、百官表、刑法志、艺文志
→ 田余庆《秦汉魏晋史探微》
→ 王子今等专题研究
```

### 用《大汉帝陵》转成读书路线

《大汉帝陵》天然按帝王顺序走，适合建立西汉政治时间线：

```text
高祖
→ 惠帝、吕后
→ 文帝
→ 景帝
→ 武帝
→ 昭帝、霍光
→ 宣帝
→ 元成哀平
→ 王莽
```

读书时不要只看皇帝个人，而要把每一代放进一个问题里。

高祖：建国问题。

```text
刘邦如何从反秦力量中胜出？
汉初如何继承和调整秦制？
异姓王为什么必须被清除？
```

吕后到文景：稳定问题。

```text
吕后政治如何维持刘氏天下？
文景之治为什么能出现？
七国之乱说明郡国并行出了什么问题？
```

武帝：扩张问题。

```text
汉武帝如何把文景积累转化为帝国扩张？
对匈奴战争、盐铁财政、均输平准如何联动？
儒术独尊和实际政治中的法家技术如何并存？
```

昭宣：中兴问题。

```text
霍光辅政为何能稳定政局？
宣帝为什么常被视为西汉政治的成熟阶段？
外戚、儒吏、法吏、边疆治理如何平衡？
```

元成哀平到王莽：衰落和改制问题。

```text
西汉后期为什么外戚、宦官、儒生政治问题突出？
土地兼并、财政、灾异政治如何影响皇权合法性？
王莽为什么能篡汉，又为什么迅速失败？
```

可以这样做笔记：

```text
帝王：
时代问题：
关键人物：
关键制度：
重大事件：
长期影响：
```

例如汉武帝：

```text
帝王：汉武帝
时代问题：帝国扩张与财政重组
关键人物：卫青、霍去病、张骞、桑弘羊、董仲舒、司马迁
关键制度：盐铁、均输、平准、察举、内外朝
重大事件：漠北之战、张骞通西域、罢黜百家、轮台诏
长期影响：塑造汉帝国边疆、财政和儒法混合政治
```


```text
“霸王道杂之”说明了什么政治现实？

```

重点篇目：


### 王莽作为深入训练

王莽是很适合作深入阅读训练的人物。他不是单纯的篡位者，也不是简单的复古理想主义者，而是西汉后期结构性问题的集中爆发点。

读王莽，要同时看五层：

```text
个人形象
→ 外戚政治
→ 儒家经学与禅让合法性
→ 西汉后期社会经济危机
→ 复古改制为何失败
```

个人形象：

```text
谦恭下士
周公形象
托孤辅政
安汉公
假皇帝
禅让
```

社会经济和改制关键词：

```text
土地兼并
奴婢问题
王田
私属
币制改革
五均六筦
盐铁酒专卖
```

失败原因可以从几类原因叠加看：

```text
改革过急，制度频繁更改
复古名义和现实社会脱节
币制混乱，损害经济信用
地方执行失控
豪强和普通百姓都不满意
边疆政策失误
黄河改道、灾荒加剧
赤眉、绿林起义爆发
```

读王莽时适合做表：

```text
问题：西汉后期土地兼并
王莽方案：王田制
理论依据：复古井田、限制兼并
现实阻力：豪强地主、行政执行、农民实际负担
结果：推行困难，反复更改，激化不满
```

一句话说：

```text
王莽不是西汉灭亡的唯一原因，
而是西汉后期外戚政治、经学合法性、土地财政危机、
复古理想和行政失灵交汇出来的结果。
```

### 专题研究

- **田余庆《秦汉魏晋史探微》**：论文集，适合进阶。《说张楚》《论轮台诏》等能训练从一个问题切入大结构。
- **王子今《秦汉交通史稿》**：看帝国如何靠道路、运输、通信、迁徙和区域交通运转。
- **《盐铁论》**：理解武帝以后财政路线、民生压力和儒法政策争论的重要材料。
- **《西汉会要》《东汉会要》**：进阶查制度用，不适合一开始通读。

### 下一步重点

现在最值得补三件事：

```text
读原典校准叙事：
重点读《史记》《汉书》相关篇章，看看视频讲述背后的原文怎么写。

读制度材料：
尤其《汉书·食货志》《百官公卿表》《刑法志》《外戚传》《王莽传》。

读现代研究：
例如吕思勉《秦汉史》、王子今《秦汉史：帝国的成立》、田余庆《秦汉魏晋史探微》。
```

一句话评价：

```text
西汉史若已有完整骨架，
下一步就要从视频叙事型知识，
升级为原典和问题驱动型知识。
```

## 四、东汉阅读路径

东汉要和西汉分开读。西汉的关键词是“帝国建立与中央集权成熟”，东汉的关键词则是：

```text
豪强社会
外戚宦官循环
士人清议
经学与名节
地方化
黄巾与州牧
```

如果已经熟悉光武帝刘秀、明帝、章帝、和帝、安帝、桓帝、灵帝这些关键帝王，就已经抓到东汉主线的骨架：从重建、稳定，到外戚宦官政治失控、党锢与黄巾前夜。

### 东汉主线

可以这样理解东汉：

```text
刘秀重建汉室
→ 光武柔道治国，依靠南阳、河北、关中等功臣与地方豪强
→ 明章之治，东汉制度相对稳定
→ 和帝以后皇帝幼弱，外戚辅政和宦官夺权循环出现
→ 安帝以后政治质量下降，羌乱、财政、外戚问题加重
→ 桓帝、灵帝时期党锢之祸，士人与宦官对立
→ 黄巾起义、州牧坐大
→ 东汉中央解体，进入军阀时代
```

### 不要只看宫廷斗争

很多人读东汉只记得外戚宦官。真正要深入，还要看地方社会变化。

读东汉时要追问：

```text
为什么东汉豪强地主这么重要？
为什么刘秀不能像西汉初年那样彻底打散地方势力？
察举、经学、门生故吏如何形成士人网络？
外戚宦官斗争为什么会反复出现？
党锢之祸为什么会成为士人政治记忆？
黄巾起义为什么能迅速扩散？
州牧和刺史为什么会变成割据力量？
```

### 分段读法

第一段：光武重建。

```text
问题：刘秀如何在新莽和军阀混战后重建汉室？
他如何处理功臣和豪强？
```

第二段：明章稳定。

```text
问题：东汉制度如何恢复？
经学、礼制、边疆、外戚力量如何安排？
```

第三段：和安以后。

```text
问题：幼主继位为什么导致外戚与宦官循环？
羌乱和财政压力如何削弱国家？
```

第四段：桓灵末世。

```text
问题：党锢之祸、士人清议、宦官专权、黄巾起义、
州牧坐大如何把东汉推向崩解？
```

### 现代书目

- **吕思勉《秦汉史》东汉部分**：看整体框架。
- **王子今《秦汉史：帝国的成立》后半**：看东汉社会和制度变化。
- **田余庆《秦汉魏晋南北朝简史》相关部分**：先打底。
- **田余庆《秦汉魏晋史探微》**：东汉到魏晋过渡很重要，适合理解士族、门阀萌芽。
- **阎步克《察举制度变迁史稿》**：深入察举、士人政治、门生故吏。
- **祝总斌《两汉魏晋南北朝宰相制度研究》**：看外朝、尚书台和相权变化。
- **余英时《东汉生死观》**：偏思想文化，理解东汉人的精神世界、神仙方术和死后观念。

### 读东汉的关键

可以用四个关键词重读东汉：

```text
豪强社会
察举士人
外戚宦官
地方割据
```

东汉一读透，魏晋门阀和三国军阀就接得上了。

## 五、前四史阅读路径

前四史包括《史记》《汉书》《后汉书》《三国志》。不建议从第一页硬读到最后一页。

推荐顺序：

```text
《史记》
→ 《三国志》
→ 《后汉书》
→ 《汉书》
```

理由：

- 《史记》叙事强，人物鲜明，最适合入门。
- 《三国志》时代背景较熟悉，配裴松之注读，细节丰富。
- 《后汉书》人物传好，士人、隐逸、党锢、宦官、外戚很有价值。
- 《汉书》更正史化，制度、经学和政治结构较重，适合后读。

## 七、魏晋南北朝阅读路径与书目

魏晋南北朝是从秦汉帝国到隋唐帝国之间最关键的“重组期”。它不只是大乱世，而是汉帝国崩解之后，士族门阀、南北对峙、北方民族政权、江南开发、北朝制度整合共同塑造隋唐前夜的过程。

可以按五步抓：

```text
汉末三国
→ 西晋短暂统一与八王之乱
→ 永嘉之乱、东晋十六国
→ 南朝宋齐梁陈与北魏统一北方
→ 东魏西魏、北齐北周，最后入隋
```

核心变化有四条：

```text
士族门阀形成并掌权
北方民族政权与胡汉融合
江南开发和南朝政治成熟
北朝制度重组，孕育隋唐
```

### 汉末三国：帝国从地方裂开

东汉末不是突然崩溃。前面已经有几层问题：

```text
豪强坐大
察举形成士人网络
外戚宦官内斗
党锢之祸打击士人
黄巾起义冲击地方
州牧刺史军事化
```

董卓进京后，中央权威实际崩掉。曹操能起来，不只是个人雄才，也因为他抓住了几个关键：

```text
挟天子以令诸侯
屯田解决军粮
整合兖豫士族与寒门军功人才
把军阀集团变成准国家机器
```

三国里，魏最像后来的北方国家机器；蜀更像以荆益集团和复汉名义支撑的偏安政权；吴则依靠江东士族、长江防线和水军。这里已经埋下后来南北对峙的影子。

### 西晋：统一太快，基础太脆

司马氏篡魏后建立西晋，灭吴统一天下，看似恢复大一统，但问题很多：

```text
司马氏得国不正，依赖宗王屏藩
门阀士族垄断政治
九品中正制固化门第
皇族诸王掌兵，中央控制力弱
```

晋武帝分封宗王，本意是防止异姓权臣，结果宗王有兵、有地、有政治资格。惠帝无能，贾后乱政，诸王争权，爆发八王之乱。

八王之乱不是普通宫廷政变，而是把西晋中央军力、财政和北方社会秩序消耗干净。之后匈奴刘渊、刘聪起兵，洛阳、长安相继陷落，晋怀帝、晋愍帝被俘，这就是永嘉之乱。北方士族大量南渡，形成“衣冠南渡”。

### 东晋：王与马共天下

司马睿在建康建立东晋，但他能立住，靠的不是司马皇权本身，而是南渡士族和江南本土力量支持。王导、王敦代表琅琊王氏，地位极重，所以有“王与马共天下”。

东晋政治不是普通皇帝专制，而是门阀共治：

```text
皇帝弱
高门士族强
王、谢、庾、桓轮流执政
荆州、扬州等军事重镇影响朝局
北府兵成为关键武力
```

几个关键节点：

- **王敦之乱**：说明大族军府可以威胁皇权。
- **桓温北伐**：既有恢复中原名义，也有个人篡位资本。
- **淝水之战**：谢安、谢玄、北府兵击败前秦，保住东晋。
- **刘裕崛起**：北府兵将领出身，最终取代司马氏，建立刘宋。

东晋要读明白，就要看：门阀士族为什么能掌权，又为什么最后被寒门武人刘裕取代。

### 十六国：北方重组

“五胡乱华”这个说法情绪很重，容易让人只看杀戮。更好的读法是：北方进入多民族政权竞争和国家重组期。

匈奴、羯、鲜卑、氐、羌等政权纷纷建立国家。最重要的几个：

```text
汉赵：刘渊、刘聪，灭西晋
后赵：石勒、石虎，羯族政权，军事强横
前秦：苻坚，短暂统一北方，淝水失败
后秦：姚苌、姚兴，关中政权，佛教兴盛
北燕、南燕、后燕：慕容鲜卑诸政权
```

苻坚尤其值得读。他不是简单昏君。前秦一度整合北方，任用王猛，治理较有效。但淝水之战失败后，前秦内部族群和地方矛盾爆裂，北方重新分裂。

十六国的关键问题：

```text
胡族军事集团如何治理汉地？
汉族士人如何进入胡族政权？
民族身份和政治合法性怎样转换？
为什么短期统一难以稳定？
```

### 十六国记忆法

原来这个顺口溜也很好：

```text
燕前燕后南北立，
西北前南后五凉。
两赵一夏一成汉，
西方前后有三秦。
```

可以稍作整理，让国名更齐：

```text
前后南北并西燕，
前后南北又西凉。
二赵一夏兼成汉，
前后西秦号三秦。
```

可以用顺口溜记国名：

```text
五燕前后南北西，
五凉前后南北西。
二赵一夏一成汉，
前后西秦是三秦。
```

如果按传统“十六国”名单，通常是：

```text
前赵、后赵、前燕、前秦、后秦、成汉、
前凉、后凉、南凉、西凉、北凉、
西秦、南燕、北燕、夏、后燕
```

也就是：

```text
二赵
三秦
五凉
四燕
一夏
一成汉
```

传统十六国通常不列西燕；如果按十六国时期政权记忆，可以把西燕也记住，称“五燕”。

国名速查：

```text
二赵：前赵、后赵
三秦：前秦、后秦、西秦
五凉：前凉、后凉、南凉、北凉、西凉
四燕：前燕、后燕、南燕、北燕
一夏：夏 / 胡夏
一成汉：成汉
另记：西燕，通常不列入传统十六国
```

地图记忆表：

```text
国号组   代表政权                  地理核心
二赵     前赵、后赵                山西、河北、中原
三秦     前秦、后秦、西秦          关中、陇右、河湟
五凉     前后南北西凉              河西走廊
诸燕     前后南北燕，另有西燕      辽西、河北、山东
成汉     成汉                      巴蜀
胡夏     夏                        陕北、关中北部
```

分组理解：

- **二赵**：前赵是匈奴刘渊、刘聪一系，先称汉，后称赵，灭西晋；后赵是羯族石勒、石虎，控制河北、中原。
- **三秦**：前秦是氐族苻氏，苻坚、王猛最重要；后秦是羌族姚氏，姚兴时佛教兴盛；西秦是鲜卑乞伏氏，在陇西、河湟活动。
- **五凉**：都在河西走廊，是中原通西域的走廊政权。前凉张氏，后凉吕光，南凉秃发氏，北凉沮渠氏，西凉李暠。
- **诸燕**：看慕容鲜卑，从辽西进入河北、山东。前燕、后燕、南燕、北燕之外，还有传统十六国不列的西燕。
- **成汉**：李特、李雄建立，巴蜀割据，后来被东晋桓温灭。
- **胡夏**：赫连勃勃建立，匈奴铁弗部，建统万城，活动于陕北、关中北部。

十六国主线：

```text
西晋崩溃
→ 二赵冲击中原
→ 前秦统一北方
→ 淝水后重新分裂
→ 北魏统一北方
```

这条线记住，再看国名就不乱。

### 南朝宋齐梁陈：江南成熟，但皇权多疑

刘裕建立宋，标志东晋门阀共治被寒门军人皇权替代。南朝皇权比东晋更强，但也更残酷、更不稳定。

南朝四代：

```text
宋：刘裕开国，元嘉之治，后期宗室内斗严重
齐：萧道成代宋，国祚短，宗室残杀频繁
梁：梁武帝前期文化繁荣，后期侯景之乱摧毁江南
陈：陈霸先重建，但国力有限，最后被隋灭
```

几个细节：

- **刘裕北伐**：曾收复长安、洛阳，但不能长期守住，因为南朝补给、骑兵和北方根基不足。
- **元嘉北伐失败**：刘义隆想恢复中原，结果北魏反击到瓜步，南朝再难北进。
- **梁武帝**：重佛教、重文化，前期政治还算稳定；晚年放纵宗室、接纳侯景，引发大乱。
- **侯景之乱**：是南朝转折点，建康被毁，江南士族和经济遭重创。
- **陈朝**：是在废墟上维持江南，不能简单归结为陈后主荒淫亡国。

南朝要看两条线：

```text
江南经济文化持续发展
皇权与士族、宗室、寒门武人之间反复冲突
```

### 北魏：北朝真正的转折

北魏是理解隋唐前史的关键。拓跋鲜卑从代北部族政权，逐步变成统治华北的大帝国。

几个节点：

- **道武帝拓跋珪**：建立北魏国家基础。
- **太武帝拓跋焘**：统一北方，灭北凉，打击柔然。
- **冯太后、孝文帝改革**：均田制、三长制、迁都洛阳、改汉姓、穿汉服、通婚、用汉制。
- **六镇之乱**：改革后代北军事贵族被边缘化，北方边镇爆发叛乱，北魏崩溃。
- **东西魏分裂**：高欢控制东魏，宇文泰控制西魏。

孝文帝改革不能简单说“汉化成功”或“汉化失败”。它确实推动北魏国家制度化，但也制造了洛阳汉化贵族与代北军事集团的裂缝。六镇之乱就是这个矛盾的大爆发。

### 北齐北周：隋唐从北周出来

东魏到北齐，高欢、高洋一系控制河北、山东，军事强，但宫廷政治混乱，北齐皇室残暴内耗严重。

西魏到北周，宇文泰在关中建立政权，实力起初弱于东魏北齐，但制度整合很强：

```text
府兵制
关陇集团
六官制度
胡汉军事贵族融合
```

北周灭北齐，统一北方。杨坚正是在北周政治军事结构中上位，建立隋朝。隋唐制度很多要从西魏北周找源头，而不是只从南朝或汉魏找。

北朝后期的关键不是“谁更文明”，而是：

```text
谁能更有效整合军事、财政、土地、贵族和官僚制度？
```

答案最终是西魏北周系统。

### 北朝五国与隋

严格说，北朝通常指五个政权：

```text
北魏
东魏
西魏
北齐
北周
```

隋朝不算北朝。它是承接北周制度、结束南北朝、重新统一天下的王朝。

关系可以这样记：

```text
北魏
→ 分裂为东魏、西魏
→ 东魏被北齐取代
→ 西魏被北周取代
→ 北周灭北齐，统一北方
→ 杨坚代北周建隋
→ 隋灭陈，结束南北朝
```

所以可以说：

```text
北朝五国，加上承北周而来的隋。
```

北朝的关键不是背朝代名，而是看清一条线：

```text
拓跋北魏统一北方
→ 孝文帝改革造成新旧裂缝
→ 六镇之乱导致东西分裂
→ 西魏北周系统完成制度整合
→ 隋唐继承北周隋的制度成果
```

如果喜欢从人物入手，可以先抓这些人：

```text
曹操、司马懿、司马炎、司马睿
王导、谢安、桓温、刘裕
刘渊、石勒、苻坚、王猛
拓跋珪、拓跋焘、冯太后、孝文帝
高欢、宇文泰、杨坚
```

每个人都不要只看传记，要追问他背后的集团：

```text

司马氏：士族政治与篡魏合法性
王导：南渡士族与东晋门阀
谢安：门阀政治与北府兵

宇文泰：关陇集团与北周制度

```

### 原典读法

不要一开始通读《晋书》《宋书》《魏书》。可以用《资治通鉴》魏晋南北朝部分串年代，再回头读正史人物传。

《资治通鉴》尤其适合串：

```text
八王之乱
永嘉之乱
祖逖北伐
淝水之战
刘裕北伐
北魏统一北方
孝文帝改革
六镇之乱
东西魏分裂
北周灭北齐
隋灭陈
```

正史篇目可以这样选：

```text
《三国志》：曹操、诸葛亮、司马懿相关
《晋书》：武帝、惠帝、元帝、王导、谢安、桓温、刘琨、祖逖
《宋书》：刘裕、文帝、谢灵运
《南齐书》：萧道成、萧衍前史

《陈书》：陈霸先、陈后主
《魏书》：道武帝、太武帝、孝文帝、崔浩
《北齐书》《周书》：高欢、宇文泰、北齐北周制度
《南史》《北史》：可作综合对照
```

### 通史骨架

- **王仲荦《魏晋南北朝史》**：首选断代通史，政治、制度、经济、民族和文化都覆盖。
- **吕思勉《两晋南北朝史》**：史料密，观点强，适合第二遍或配合王仲荦读。
- **万绳楠《魏晋南北朝史论稿》**：专题意识更强，适合有基础后读。

### 门阀、士族、制度

- **田余庆《东晋门阀政治》**：理解“王与马共天下”、侨姓士族、北府兵和东晋政治结构的核心书。
- **田余庆《拓跋史探》**：理解北魏和拓跋鲜卑如何从部族走向国家。
- **李凭《北魏平城时代》**：重点看北魏前期、平城政权、拓跋文化与汉文化交错。
- **张金龙《北魏政治史》**：北魏政治研究很细，适合查专题，不建议一开始通读。
- **宫崎市定《九品官人法研究：科举前史》**：理解九品中正制如何支撑士族政治。
- **川胜义雄《六朝贵族制社会研究》**：从社会结构看六朝贵族制。
- **唐长孺《魏晋南北朝史论丛》**：制度、土地、兵制、民族、宗教、玄学等专题都很强。
- **周一良《魏晋南北朝史论集》**：适合专题精读。
- **陈寅恪《隋唐制度渊源略论稿》**：看北朝制度如何进入隋唐。
- **陈寅恪《唐代政治史述论稿》**：关陇集团、北周到隋唐的政治脉络很关键。

### 思想、宗教、文学

- **余嘉锡《世说新语笺疏》**：读魏晋风度、清谈、士族生活的好本子。
- **汤用彤《汉魏两晋南北朝佛教史》**：佛教进入中国、与玄学互动、士人接受佛教的经典书。
- **罗宗强《魏晋南北朝文学思想史》**：串联建安、正始、太康、玄言、山水、宫体等文学变化。
- **罗新《漫长的余生：一个北魏宫女和她的时代》**：以北魏宫女王钟儿的命运切入孝文帝前后北魏政治，适合从故事感转向史料意识。

### 北朝原典

北朝原典可以这样用：

```text
《魏书》：北魏主史料
《北齐书》：北齐
《周书》：西魏、北周
《北史》：综合北朝人物
《资治通鉴》：串北魏、东西魏、北齐北周事件线
```

### 推荐顺序

```text
王仲荦《魏晋南北朝史》
→ 《资治通鉴》晋纪、宋齐梁陈纪、北朝相关段
→ 田余庆《东晋门阀政治》
→ 田余庆《拓跋史探》
→ 李凭《北魏平城时代》
→ 宫崎市定《九品官人法研究》
→ 陈寅恪《隋唐制度渊源略论稿》
→ 唐长孺、周一良论文集
→ 川胜义雄《六朝贵族制社会研究》
→ 《世说新语笺疏》、汤用彤、罗宗强
```

一句话总结：魏晋南北朝要读成“从汉帝国崩解到隋唐帝国再造”的过程。东晋南朝看门阀和江南，十六国北朝看胡汉融合和国家重建，最后北周隋唐看制度整合。魏晋南北朝不是中国历史的断裂层，它是隋唐帝国的熔炉。

## 八、隋唐史阅读路径

隋唐史很适合接在魏晋南北朝后面读。它不是孤立的新朝代，而是北朝制度、关陇集团、门阀士族、胡汉关系和南北文化长期融合后的结果。

如果手里有《旧唐书》《新唐书》，不建议直接从头硬读。更稳的路径是：

```text
通史入门
→ 《资治通鉴》串时间线
→ 《旧唐书》《新唐书》读本纪和列传
→ 志书和制度专题
→ 陈寅恪、岑仲勉等专题研究
```

### 关键主线

读隋唐史时，可以先抓这些问题：

- **制度承接**：从九品中正、门阀政治，走向科举、三省六部、均田租庸调。
- **关陇集团**：北朝到隋唐建国的核心政治军事实力。
- **皇权与贵族**：从高祖、太宗到武后、中宗、玄宗，皇权如何压制、改造士族。
- **边疆与军事**：府兵制、节度使、安西北庭、突厥、吐蕃、回鹘。
- **安史之乱**：唐史最大分水岭，前后几乎是两个唐朝。
- **藩镇、宦官、朋党**：中晚唐政治的三大难题。
- **财政转型**：从租庸调到两税法，国家治理方式发生改变。
- **文学与士人**：科举、幕府、贬谪、边塞、古文运动，都和制度环境有关。

### 从影视印象转入正史

如果之前看过《隋唐演义》《大唐帝陵》《贞观之治》《唐明皇》，这些可以作为入口，但读史时要有校准意识。

《隋唐演义》是演义系统，秦琼、程咬金、罗成、单雄信等人物有文学魅力，但不能当作事实框架。隋末真正要抓的是：

- 隋炀帝统治失衡。
- 关东豪强与地方割据。
- 李渊集团如何从太原起兵到统一。
- 李密、王世充、窦建德为什么失败。

《大唐帝陵》适合建立帝王世系和陵寝、政治记忆的关系。可以每看一位帝王，就回去读《旧唐书》或《新唐书》本纪，再读他身边三五个关键人物传。

《贞观之治》适合入门贞观政治，但正史里不要只看“明君贤臣”，还要看：

- 玄武门之后的合法性重建。
- 太宗如何使用山东士族、关陇旧臣、隋臣和功臣集团。
- 谏诤政治背后的君臣权力结构。
- 对突厥、高昌、高丽等边疆政策。
- 太子问题和后期政治阴影。

《唐明皇》可以转入开元天宝和安史之乱。读史时要把“爱情悲剧”放小，把制度变化放大：

- 开元前期为什么能治。
- 张九龄罢相意味着什么。
- 李林甫专政如何改变相权和用人。
- 节度使坐大为什么不是安禄山一个人的问题。
- 府兵制瓦解、募兵制兴起、财政压力和边防扩张。
- 安史之乱后唐朝为什么没有真正恢复到开元状态。

### 《旧唐书》《新唐书》的用法

大致可以这样分工：

- **《旧唐书》**：较接近原始史料，保存材料多，叙事有时更朴实。
- **《新唐书》**：文字更修整，志表较强，但议论和删改也较重。
- **《资治通鉴》唐纪**：最适合串联事件，尤其玄武门、武周、安史、藩镇、宦官这些复杂线索。

可以先读《旧唐书》本纪、重要列传，再用《新唐书》补制度、表志和人物评价，最后拿《资治通鉴》串时间。

### 正规深入读历史的方法

深入读历史，不只是知道故事，而是要从人物好恶转为制度、结构和史料互证。

每读一章或一传，可以固定写五项：

```text
时间：
人物：
事件：
制度背景：
历史影响：
```

再加一句自己的判断：

```text
我认为这一事件的关键原因是……
```

不要大段摘抄。能复述，能解释，才算读进去。

## 九、隋唐史专题研究书目

专题研究不必一上来全读，要按问题线索选择。

### 先读这一组

- **岑仲勉《隋唐史》**：隋唐通史，但写法偏研究型，适合做主干。
- **陈寅恪《隋唐制度渊源略论稿》**：看隋唐制度从北朝、南朝、魏晋以来如何演变。
- **陈寅恪《唐代政治史述论稿》**：看关陇集团、关中本位政策、武周革命、党派分野和外族盛衰。
- **黄永年《唐史十二讲》**：适合作专题入门，涉及安史之乱、两税法、永贞革新、武则天、唐玄宗、唐宪宗等。

推荐先拿下这几本。它们分别提供通史骨架、制度源流、政治史识和专题入口。

### 岑仲勉相关

- **岑仲勉《唐史余沈》**：唐史考证札记，适合读《新旧唐书》《通鉴》时查疑难问题，不建议从头硬读。
- **岑仲勉《通鉴隋唐纪比事质疑》**：校读《资治通鉴》隋唐部分，考辨史事。
- **岑仲勉《府兵制度研究》**：读府兵制、均田制和唐初军事结构。

### 制度专题

- **谷霁光《府兵制度考释》**：可与岑仲勉《府兵制度研究》对读。
- **张国刚《唐代官制》**：看三省六部、使职差遣和唐代官僚运作。
- **严耕望《唐代交通图考》**：史地、交通、州府军镇和边疆路线，非常扎实，适合进阶查读。

### 政治社会专题

- **毛汉光《中国中古社会史论》**：从士族、门阀、社会阶层看汉末到唐亡的变化，适合打通魏晋到隋唐。
- **黄永年《唐代史事考释》**：偏考证，适合进阶。

### 外部视角

- **《剑桥中国隋唐史：589-906年》**：西方汉学视角，结构清楚，适合与国内学者对读。

### 推荐顺序

```text
岑仲勉《隋唐史》
→ 《资治通鉴》唐纪重点段
→ 陈寅恪《隋唐制度渊源略论稿》
→ 陈寅恪《唐代政治史述论稿》
→ 黄永年《唐史十二讲》
→ 岑仲勉《唐史余沈》《通鉴隋唐纪比事质疑》
→ 兵制、官制、交通、社会史专题
```

## 十、五代十国阅读路径

五代十国不是“乱世故事合集”，而是晚唐藩镇、宦官、财政崩坏之后，通向宋代中央集权的中间层。

读五代史，要抓四条主线：

- **武人政治**：节度使、牙兵、禁军、军阀更替。
- **财政重组**：藩镇财权、盐铁、两税、地方割据。
- **南北差异**：北方五代政权短促更替，南方十国相对稳定经营。
- **宋代前夜**：赵匡胤为什么能结束五代，宋为何重文抑武、强干弱枝。

可以用这首口诀先记住国名和地理：

### 从人物入手是否可行

从朱温、李克用、李存勖、郭威这些人物入手，是很好的入口。五代这种时代，政权更替往往由少数强人、军队集团和地方资源推动。

但要避免停留在人物传奇。深入读法应当从人物往外扩三层：

```text
人物经历
→ 他背后的集团
→ 他所处的制度与资源结构
→ 他造成的历史后果
```

例如读朱温，不只看他从黄巢部将到篡唐，还要问：

- 他为什么能从汴州发展起来？
- 宣武军的地理和财政条件如何？
- 他如何吞并或压制其他藩镇？
- 他篡唐为什么成功？
- 后梁为什么很快陷入合法性和军事困境？

读李克用，不只看“独眼龙”“沙陀英雄”，还要问：

- 沙陀集团怎样进入唐末政治？
- 河东为什么能成为长期基地？
- 他和朱温的冲突，背后是个人仇怨，还是两种军事集团竞争？
- 河东集团为什么能延续到李存勖？

读李存勖，不只看他灭梁、好伶人、败亡，还要问：

- 他为什么能灭后梁？
- 灭梁后为什么不能完成政治整合？
- 后唐为何继承强大军事力量，却迅速内乱？
- 庄宗败亡和伶人、宦官、功臣、财政之间有什么关系？

读郭威，不只看他代汉称帝，还要问：

- 后汉为什么失控？
- 郭威如何掌握禁军和朝廷中枢？
- 后周改革为什么能为柴荣、赵匡胤铺路？
- 郭威、柴荣与宋初制度有什么连续关系？

### 人物研究笔记模板

每读一个人物，可以这样做笔记：

```text
人物：
时代位置：
出身与集团：
根据地：
主要军事力量：
财政来源：
关键盟友：
主要敌人：
重大转折：
失败或成功原因：
后续影响：
可入诗文的形象或典故：
```

例如朱温：

```text
人物：朱温
时代位置：唐末到后梁建立
出身与集团：黄巢旧部，后归唐，宣武军节度使
根据地：汴州
主要军事力量：宣武军、兼并后的中原藩镇兵力
财政来源：汴梁中原交通与州县财赋
关键盟友：唐廷名义、部分藩镇、降将
主要敌人：李克用河东集团、关中残余力量、淮南杨行密
重大转折：降唐、据汴、挟昭宗、杀宦官、篡唐
成功原因：控制中原要地，军事现实主义强，善于兼并
失败隐患：合法性薄弱，家族继承混乱，与河东长期对抗
后续影响：开启五代，后梁成为后唐主要对手
```

这样就不是普通人物传记阅读，而是在用人物做切口研究时代。

### 《旧五代史》《新五代史》的用法

- **《旧五代史》**：材料保存更多，纪传体较接近事实底本。
- **《新五代史》**：欧阳修笔法、褒贬和义例很强，文学性与史论性高，但材料取舍明显。

两书适合对读：先用《旧五代史》看事实，再看《新五代史》如何评价同一人物和事件。

不要一开始通读，可以先读这些人物线：

```text
朱温
李克用
李存勖
石敬瑭
刘知远
郭威
柴荣
冯道
王建
孟知祥
杨行密
钱镠
李昪
李煜
```

这些人基本能串起梁、唐、晋、汉、周和主要十国。

### 配《资治通鉴》读

《资治通鉴》到后周显德六年，即 959 年为止。五代部分特别适合用它串线，因为事件更替很快，单读纪传体容易碎。

重点读：

```text
黄巢余波与朱温崛起
朱温篡唐
后梁与河东集团对抗
李存勖灭梁与后唐兴亡
石敬瑭割燕云
契丹南下与后晋灭亡
郭威代汉
柴荣改革与北伐
```

《通鉴》不覆盖北宋建立之后的十国收束，所以南方十国后段要靠《十国春秋》或相关通史补。

### 北方主线与南方支线

北方五代主线：

```text
晚唐藩镇
→ 朱温篡唐
→ 后梁后唐争霸
→ 沙陀三朝
→ 石敬瑭与燕云
→ 契丹南下
→ 郭威、柴荣改革
→ 宋代建国
```

南方十国支线：

```text
吴越：钱镠与东南经营
南唐：李昪、李璟、李煜
前后蜀：西川割据与文化
南汉：岭南政权
楚、闽、荆南：地方割据样态
```

每读一个政权，固定问四个问题：

```text
它靠什么军事力量立国？
它的钱粮从哪里来？
它和中原、契丹、邻国是什么关系？
它为什么能维持，或为什么灭亡？
```

这样读，五代就不会只是“你杀我、我杀你”的混乱叙事，而会变成一段从唐到宋的制度转型史。

## 十一、五代十国书目

如果已经有《旧五代史》《新五代史》，下一步重点是补三类书：串线书、结构解释书、十国材料书。

### 原典与串线

- **薛居正等《旧五代史》**：事实底本。
- **欧阳修《新五代史》**：看史论、笔法和义例。
- **司马光《资治通鉴》五代部分**：按年月串联朱温、李克用、李存勖、石敬瑭、郭威、柴荣这条线。
- **吴任臣《十国春秋》**：十国史料总集，进阶查读用，不建议从头通读。

### 通史主干

- **王仲荦《隋唐五代史》**：把唐末到五代放在连续框架里看，不把五代当孤立乱世。
- **吕思勉《隋唐五代史》**：史料感强，适合进阶读，但文言和史料摘引较多。

### 结构与制度研究

- **王赓武《五代时期北方中国的权力结构》**：解释五代北方政权如何从藩镇武人政治，逐步走向重新建立中央权威。很适合从朱温、李克用、郭威等人物切入后继续深入。
- **杜文玉《乱世规绳：五代十国制度研究》**：看五代十国制度，涉及贡举、选官、考课、职官、军事等。
- **杜文玉《五代十国经济史》**：理解南方十国为什么相对稳定，以及吴越、南唐等政权如何经营地方。
- **杜文玉《南唐史略》**：如果对李昪、李璟、李煜和南唐感兴趣，可以单独看南唐线。

### 衔接宋史

五代十国的意义之一，是解释宋代制度为何如此重文抑武、强干弱枝。读完五代，可以接宋史入门书，尤其关注：

- 宋初禁军制度。
- 中央集权与地方削权。
- 科举扩大与士大夫政治。
- 财政、盐铁、转运与地方治理。

### 推荐顺序

```text
《资治通鉴》五代部分
→ 《旧五代史》梁、唐、晋、汉、周本纪
→ 《旧五代史》朱温、李克用、李存勖、石敬瑭、郭威、柴荣、冯道等传
→ 《新五代史》对应篇章
→ 王赓武《五代时期北方中国的权力结构》
→ 王仲荦或吕思勉《隋唐五代史》五代部分
→ 《十国春秋》按需查南方十国
```

人物经历是入口，背后的藩镇、军队、财赋和合法性才是深入处。

## 十二、宋辽夏金阅读路径与书目

宋朝不能只按“积贫积弱”或“文化繁荣”来读。它承接五代十国，面对辽、西夏、金、蒙古等北方政权，最终进入元朝统一天下的格局。读宋史时必须把北宋和南宋分开，同时把辽、金、西夏放进同一张地图。

### 总体主线

可以这样抓：

```text
北宋结束五代十国
→ 强干弱枝、重文抑武，建立文官政治
→ 辽宋对峙，澶渊之盟形成长期均势
→ 西夏立国，宋夏战争与边疆财政压力
→ 王安石变法，北宋制度调整失败与党争加剧
→ 金灭辽，宋金海上之盟
→ 靖康之变，北宋灭亡
→ 南宋建立，宋金长期对峙
→ 岳飞、韩世忠、秦桧与绍兴和议
→ 宋孝宗隆兴北伐、开禧北伐等恢复尝试
→ 蒙古兴起，灭西夏、灭金
→ 宋蒙联合灭金后转为宋蒙战争
→ 襄阳失守、临安降元、崖山之后元统一天下
```

一句话说：

```text
北宋看制度建构与北方压力；
南宋看偏安重建、财政军事和宋金宋蒙对峙；
辽夏金元看北方民族政权如何参与中国历史的重组。
```

### 北宋关键节点

北宋从五代十国中来，最先要解决的是武人割据问题。

重点看：

```text
陈桥兵变
杯酒释兵权
禁军、枢密院、三衙
强干弱枝
科举扩大
文官政治
澶渊之盟
庆历新政
王安石变法
元祐更化
新旧党争
海上之盟
靖康之变
```

北宋问题不只是军事弱，而是它有意压制地方和武人力量，以换取中央集权和内部稳定。这个选择解决了五代问题，也制造了对外军事和财政上的长期压力。

### 南宋关键节点

南宋不是简单“半壁江山”。它在江南重建了一个高度财政化、商业化、士大夫化的国家。

重点看：

```text
赵构南渡
建炎南渡
苗刘兵变
岳飞、韩世忠、张俊、刘光世
绍兴和议
秦桧专权
宋孝宗恢复尝试
隆兴北伐
开禧北伐
史弥远专权
蒙古灭金
宋蒙战争
襄阳之战
临安降元
崖山海战
```

南宋要抓的问题：

```text
南宋如何在江南重建国家？
为什么抗金和议和反复拉扯？
江南财政、海贸、纸币、盐茶如何支撑国家？
士大夫政治为何更加成熟，也更加内耗？
襄阳为什么成为宋蒙战争的关键？
```

### 辽、西夏、金

这三个政权不能只当宋朝的“外敌”来读，它们本身也是成熟国家。

辽：

```text
契丹建国
南北面官制
燕云十六州
辽宋澶渊之盟
辽朝多民族统治
被金与宋夹击而亡
```

西夏：

```text
党项兴起
李元昊称帝
宋夏战争
河西、陕甘、宁夏地缘
夹在宋、辽、金、蒙古之间
最终被蒙古灭亡
```

金：

```text
女真兴起
完颜阿骨打建国
灭辽
靖康之变灭北宋
宋金对峙
海陵王南侵
金世宗大定之治
蒙古崛起后金朝衰亡
宋蒙联合灭金
```

读这几个政权时要问：

```text
它的军事基础是什么？
它如何治理农耕区和草原、山地、边疆地区？
它如何吸收汉制，又保留本族制度？
它与宋的关系是战争、贸易、岁币、册封还是均势？
```

### 蒙古兴起与宋亡

元不是突然出现的。蒙古兴起后，先后改变了整个东亚和欧亚政治格局。这里先把它作为宋金格局的结尾；元朝本身另列一章。

主线：

```text
铁木真统一蒙古诸部
成吉思汗建蒙古帝国
灭西夏
蒙古与南宋联合灭金
窝阔台、蒙哥时期继续扩张
忽必烈建立元朝
襄阳失守
临安降元
崖山之后南宋灭亡
元统一天下
```

读元统一要看三层：

```text
草原帝国的军事组织
对中原和江南财政资源的吸收
从蒙古帝国到元朝中国的制度转型
```

### 原典读法

不要一开始通读《宋史》《辽史》《金史》《元史》。可以先用《续资治通鉴》或通史串线，再按人物和制度回到正史。

宋史篇目可先抓：

```text
《宋史·太祖本纪》
《宋史·太宗本纪》
《宋史·仁宗本纪》
《宋史·神宗本纪》
《宋史·徽宗本纪》
《宋史·钦宗本纪》
《宋史·高宗本纪》
《宋史·孝宗本纪》
《宋史·岳飞传》
《宋史·韩世忠传》
《宋史·王安石传》
《宋史·司马光传》
《宋史·秦桧传》
《宋史·文天祥传》
```

辽金元可先抓：

```text
《辽史·太祖本纪》《辽史·圣宗本纪》
《金史·太祖本纪》《金史·世宗本纪》
《元史·太祖本纪》《元史·世祖本纪》
```

如果读制度，再看：

```text
《宋史·职官志》
《宋史·兵志》
《宋史·食货志》
《宋史·选举志》
《辽史·百官志》
《金史·百官志》
《元史·百官志》
```

### 入门通史

- **陈振《宋史》**：宋史断代通史，适合作主干。
- **虞云国《细说宋朝》**：叙述性较强，适合从零散历史节点进入宋代整体。
- **邓小南《祖宗之法：北宋前期政治述略》**：理解北宋政治传统、制度惯性和“祖宗之法”。
- **《剑桥中国宋代史》**：外部视角，适合和国内研究对读。

### 北宋与变法专题

- **邓广铭《北宋政治改革家王安石》**：读王安石变法的经典传记式研究。
- **邓小南《宋代文官选任制度诸层面》**：看文官政治、选任制度和士大夫国家。
- **漆侠《宋代经济史》**：理解宋代财政、商业、赋税、城市和经济结构。
- **包伟民《宋代地方财政史研究》**：进阶看地方财政和国家能力。

### 南宋专题

- **邓广铭《岳飞传》**：了解岳飞、南宋初期抗金和绍兴和议背景。
- **王曾瑜《宋高宗》或相关宋史著作**：理解南宋建立和高宗朝政治。
- **何忠礼《南宋政治史》**：南宋政治主线，适合进阶。
- **寺地遵《南宋初期政治史研究》**：研究南宋初期政局和权力结构。

### 辽金西夏与元

- **脱脱等《辽史》《金史》《宋史》**：正史材料，不建议一开始通读。
- **刘浦江《辽金史论》**：辽金史进阶研究。
- **姚大力《北方民族史十论》**：理解北方民族政权和中国历史结构。
- **杉山正明《疾驰的草原征服者：辽西夏金元》**：以欧亚视野看辽夏金元，适合扩展视野。
- **张帆《元史十五讲》**：元史入门较好，可放到下一章系统读。
- **韩儒林《元朝史》**：元史通史经典，可放到下一章系统读。
- **陈高华《元史研究》**：元史专题研究，可放到下一章系统读。

### 推荐顺序

```text
虞云国《细说宋朝》或陈振《宋史》
→ 《续资治通鉴》或宋辽金元通史选读
→ 邓小南《祖宗之法》
→ 邓广铭《北宋政治改革家王安石》《岳飞传》
→ 漆侠《宋代经济史》
→ 辽、西夏、金专题书
→ 张帆《元史十五讲》
→ 韩儒林《元朝史》或陈高华元史专题
```

读宋辽夏金元时，最好不要只站在宋的角度。宋、辽、西夏、金、蒙古和元共同构成这一时期的中国历史。北宋不是唐的简单延续，南宋也不是北宋的残缺版；它们是五代之后重新组织国家、又在多政权竞争中不断调整的两个阶段。

## 十三、元史阅读路径与书目

元史不能只当作“蒙古灭宋之后的一段”。它的关键，是草原帝国、中原王朝、江南财政和欧亚交通被放进同一个统治结构里。

### 元史主线

```text
铁木真统一蒙古诸部
→ 成吉思汗建立蒙古帝国
→ 窝阔台、贵由、蒙哥时期继续扩张
→ 忽必烈与阿里不哥争位
→ 忽必烈建元，定都大都
→ 灭南宋，完成统一
→ 行省制度、驿站、海运、漕运
→ 多族群统治与儒士、色目人、蒙古贵族
→ 元中后期财政、皇位、权臣和地方问题
→ 红巾军起义
→ 朱元璋崛起，元顺帝北走
```

读元史要抓几个问题：

```text
蒙古帝国如何转型为元朝？
忽必烈为什么要建立中原式王朝？
行省制度如何改变地方治理？
江南财政如何支撑大都和北方军事？
族群秩序、科举、儒士和蒙古贵族如何共处？
元末为什么财政、河患、民变和权力斗争一起爆发？
```

### 原典读法

《元史》成书仓促，材料价值大，但体例和文字问题也多。不要一开始通读。

可以先读：

```text
《元史·太祖本纪》
《元史·世祖本纪》
《元史·成宗本纪》
《元史·顺帝本纪》
《元史·耶律楚材传》
《元史·伯颜传》
《元史·阿合马传》
《元史·脱脱传》
《元史·河渠志》
《元史·百官志》
《元史·食货志》
```

### 入门与通史

- **张帆《元史十五讲》**：元史入门首选之一，结构清楚，适合先建立问题意识。
- **韩儒林《元朝史》**：元史通史经典，适合系统读。
- **《剑桥中国辽西夏金元史》相关章节**：外部视角，适合对读。

### 专题研究

- **萧启庆《内北国而外中国：蒙元史研究》**：蒙元史代表性论文集，适合进阶。
- **陈高华《元史研究》**：元史专题研究，可配合原典查问题。
- **张帆《元代宰相制度研究》**：看元代中枢制度和相权。
- **姚大力《北方民族史十论》**：从北方民族史角度理解辽金元和蒙古帝国。
- **杉山正明《疾驰的草原征服者：辽西夏金元》**：以欧亚视野看辽夏金元，适合拓宽视角。

### 推荐顺序

```text
张帆《元史十五讲》
→ 韩儒林《元朝史》
→ 《元史》本纪、志书和重点列传
→ 萧启庆《内北国而外中国》
→ 陈高华、张帆等专题研究
```

元史读懂后，明史的很多制度反应会更清楚：朱元璋为什么警惕丞相、地方割据、财政失控和军队失控，都与元末经验有关。

## 十四、明史阅读路径与书目

明史体量很大，不能只看朱元璋、朱棣、万历、崇祯几个皇帝。它的核心是：皇权高度集中、官僚制度复杂化、财政白银化、边疆和海疆压力并存，最后在内外危机中崩解。

### 明史主线

```text
朱元璋建国
→ 洪武制度，废丞相，强化皇权
→ 靖难之役，永乐迁都北京、北征、下西洋
→ 仁宣之治
→ 土木堡之变，明中期转向
→ 内阁、司礼监、厂卫政治
→ 嘉靖大礼议与倭寇
→ 张居正改革
→ 万历怠政、三大征、辽东危机
→ 东林党争
→ 女真兴起，后金建国
→ 财政崩坏、农民战争
```

清史要分两段看：前期是多民族帝国的扩张和制度整合，后期是传统帝国面对近代世界秩序的危机和转型。

## 十六、晚清近代史阅读路径与书目

晚清最好单独读，因为鸦片战争以后，已经不是传统王朝史读法，而是国际秩序、财政军事、制度改革、地方社会和民族国家转型一起出现。

```text

→ 地方团练和湘淮军
→ 第二次鸦片战争
→ 洋务运动
→ 中法战争
→ 甲午战争
→ 戊戌变法
→ 义和团和八国联军
→ 清末新政
→ 预备立宪
→ 辛亥革命
```

## 十八、一个长期节奏

读书和写作不要只囤书目，也不要只凭灵感。可以定一个小而稳定的节奏：

```text
每日读一点原典
每日抄几句好诗文
每周写一首诗或一阕词
每月精读一篇史传
```

写诗词时，先求通顺、真切、有来历，再求工稳。典故不要急着堆，用得准、用得轻，比用得多更好。格律也一样，先守正格，熟了再谈拗救和变体。

读史和写诗最终都不是为了显得古雅，而是为了看人、看世、看自己时多一点纵深。

学问无涯，但人可以有自己的舟。

```text
浩瀚史海一扁舟，
三皇五帝到商周。
春秋战国人名乱，
秦汉风云满纸留。
魏晋南北分又合，
隋唐五代几回秋。
宋元明清接民国，
一壶茶里看沉浮。
```

## 附章、诗词格律、词律与典故积累

### 从一句话说起

《庄子·养生主》有言：

> 吾生也有涯，而知也无涯。以有涯随无涯，殆已。

意思是：人的生命有限，而知识无限；若用有限生命去追逐无限知识，容易疲惫而失其要领。

这句话不是反对求知，而是提醒人：读书、治学、写作，都要有取舍，有方向。

可以借它化成几句小诗：

```text
有涯身作渡，
无涯知作潮。
不争穷海尽，
但借一灯高。
```

### 诗体大略

常见诗体可以这样理解：

- **古体诗**：又称古风，包括五古、七古、杂言古诗。句数、押韵和平仄较自由，不严格讲粘对。
- **近体诗**：唐代成熟的格律诗，包括绝句、律诗、排律。讲平仄、押韵、粘对，对律诗还讲对仗。
- **词**：又称长短句，按词牌填写。每个词牌规定字数、句式、平仄和韵位。
- **曲**：元曲、散曲一类，按曲牌写，口语性更强，可以加衬字。
- **骚体**：受《楚辞》影响，多用“兮”，句式舒展。
- **乐府诗**：本与音乐机关“乐府”有关，后也指仿乐府题材和风格的诗。
- **歌行体**：古体诗的一种发展，篇幅较长，句式流动，如《琵琶行》《长恨歌》。
- **现代诗/新诗**：白话诗，不依传统格律，重意象、节奏、分行和语言张力。
- **散文诗**：介于诗和散文之间，不一定分行，但有诗的意象和节奏。

### 平水韵与中华新韵

### 平水韵

平水韵是近体诗常用的传统押韵系统，依据中古音分韵，通常说有一百零六个韵部，分平、上、去、入四声。

写律诗、绝句时，传统上多用平水韵。一般要求：

- 通常押平声韵。
- 一韵到底，不随意换韵。
- 首句可押可不押。
- 偶句通常押韵。
- 还要配合平仄、粘对和对仗。

### 中华新韵

中华新韵按现代普通话读音分韵，通常分十四个韵部。普通话一、二声为平，三、四声为仄，不再单列古入声。

用中华新韵写旧体诗可以，但最好注明，例如：

```text
七绝·某题（中华新韵）
```

### 七绝、七律基本规则

### 七绝

七绝四句，每句七字，共二十八字。通常第 2、4 句押平声韵，首句可押可不押，第 3 句一般不押韵。

七绝平起首句入韵：

```text
平平仄仄仄平平，
仄仄平平仄仄平。
仄仄平平平仄仄，
平平仄仄仄平平。
```

七绝平起首句不入韵：

```text
平平仄仄平平仄，
仄仄平平仄仄平。
仄仄平平平仄仄，
平平仄仄仄平平。
```

七绝仄起首句入韵：

```text
仄仄平平仄仄平，
平平仄仄仄平平。
平平仄仄平平仄，
仄仄平平仄仄平。
```

### 七律

七律八句，每句七字，共五十六字。通常第 2、4、6、8 句押同一平声韵，首句可押可不押。中间两联，即第 3、4 句和第 5、6 句，通常要求对仗。

七律平起首句入韵：

```text
平平仄仄仄平平，
仄仄平平仄仄平。
仄仄平平平仄仄，
平平仄仄仄平平。
平平仄仄平平仄，
仄仄平平仄仄平。
仄仄平平平仄仄，
平平仄仄仄平平。
```

七律平起首句不入韵：

```text
平平仄仄平平仄，
仄仄平平仄仄平。
仄仄平平平仄仄，
平平仄仄仄平平。
平平仄仄平平仄，
仄仄平平仄仄平。
仄仄平平平仄仄，
平平仄仄仄平平。
```

七律仄起首句入韵：

```text
仄仄平平仄仄平，
平平仄仄仄平平。
平平仄仄平平仄，
仄仄平平仄仄平。

平平仄仄平平仄，
仄仄平平仄仄平。
```

### 几个格律概念

### 粘对

粘对是近体诗平仄安排里的两个关系。

**对**：同一联中，上句和下句的关键位置平仄相反，主要看第 2、4、6 字。

例如：

```text
仄仄平平平仄仄
平平仄仄仄平平
```

第 2、4、6 字分别相反，这叫“对”。

**粘**：上一联的下句和下一联的上句，关键位置平仄相同，也主要看第 2、4、6 字。

简单记：

```text
一联之内要相反，叫对；
两联之间要相同，叫粘。
```

### 拗救

近体诗中，某字平仄偏离标准，叫“拗”；在本句或对句的固定位置用相应平仄补救，叫“救”。

拗救不是任意破格，而是传统认可的变通。初学时最好先按标准格式写，熟悉后再考虑拗救。

### 孤平

孤平通常指在一个平收句中，除韵脚之外只剩一个平声字。

例如标准句：

```text
仄仄平平仄仄平
```

如果第三字由平变仄，成：

```text
仄仄仄平仄仄平
```

除末字韵脚外，只剩第四字一个平声，就可能犯孤平。常见补救是第五字改平：

```text
仄仄仄平平仄平
```

### 三平尾

三平尾，也常叫三平调，指一句诗最后三个字连续都是平声。

例如：

```text
……平平平
```

近体诗中通常要避免句尾三平。句中三个平声连在一起不一定都按三平尾论，最忌的是句末三平。

### 宋词与词律

宋词不严格遵循平水韵。更准确地说：

```text
诗用平水韵，词多用词韵。
```

填词时主要依据词牌。每个词牌规定：

- 字数。
- 句数。
- 句读。
- 平仄。
- 韵位。
- 押平韵还是仄韵。

传统填词多用《词林正韵》，不是直接按平水韵逐部押。

### 词牌韵法的特点

- 押韵位置由词牌决定。
- 可押平韵，也可押仄韵。
- 同一词牌可能有平韵体、仄韵体或不同变体。
- 有些词牌一韵到底，有些可以换韵。
- 词韵传统上比诗韵宽。
- 韵脚声情重要：平韵多舒缓，仄韵多顿挫。

### 正体与变体

**正体**：某个词牌最常用、最典型、被词谱作为标准的一种格式。

**变体**：同一词牌在历代作品中出现的其他被承认格式。可能多一字、少一字，某句平仄略异，韵位略有变化，或句读不同。

变体不是随便写错，必须有古人作品依据或被词谱收录。

### 诗律、词律书目

### 入门

- 王力《诗词格律》或《诗词格律概要》：建立诗韵、平仄、绝句、律诗、对仗、词牌和词韵框架。
- 龙榆生《唐宋词格律》：按词牌列格律、韵位和例词，适合边查边写。

### 进阶

- 启功《诗文声律论稿》：讲声律、平仄和诗文节奏，不只是背格式。
- 龙榆生《词学十讲》：讲选调、选韵、声情关系。
- 王力《汉语诗律学》：比《诗词格律》深入很多，适合进阶。

### 工具书

- 《平水韵》《佩文诗韵》：写律诗、绝句查韵用。
- 戈载《词林正韵》：填词查词韵用。
- 《钦定词谱》：查词牌正体、变体和韵位。
- 万树《词律》：进阶查词牌用。

### 典故积累

### 工具书

- 《中国典故大辞典》：查典故出处、原义和后世用法。
- 《汉语大词典》：查词义和书证，很适合确认诗词语汇来源。
- 商务印书馆《古代汉语词典》：日常读古文够用。
- 《汉语成语大词典》：查成语典故。

### 原典来源

典故不能只背词条，最好回到原典语境。优先读：

- 《左传》：春秋政治、战争和人物典故。
- 《史记》：历史人物典故核心来源。
- 《战国策》：纵横家故事，语言有锋芒。
- 《世说新语》：魏晋人物、清谈和风度典故。
- 《庄子》：哲理、寓言和意象典故。
- 《楚辞》：香草美人、忠怨流放和神话意象。
- 《汉书》《后汉书》《三国志》：进阶后再读。

### 蒙学类

- 《声律启蒙》：练声韵、对仗、词类对应和传统意象。
- 《笠翁对韵》：与《声律启蒙》并读，强化对仗感。
- 《训蒙骈句》：偏骈偶句式。
- 《幼学琼林》：传统常识、典故、名物、制度和人伦。
- 《龙文鞭影》：典故密度高，适合作典故索引，但不宜只背不查原典。

《声律启蒙》和《龙文鞭影》的区别：

```text
《声律启蒙》重声韵、对仗、意象组合；
《龙文鞭影》重人物故事、历史典故、蒙学记忆。
```
